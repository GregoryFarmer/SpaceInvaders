# Space Invaders
Do you remember the classic 1978 shoot-em-up arcade game Space Invaders? I recreated it using Electron.js! This game features the classic gamemode and a leaderboard system for people (albeit locally) to compete with each other! 

# Installation
To get started, please run the following commands.
> [!NOTE]
> This officially has support for Windows operating systems; other operating systems may work, but will be unofficially supported.

## Step 1.) 
```npm install```

This command installs the dependencies required for the game.
## Step 2.)
```npm run make```

This command will create a ./out directory which will contain your setup files. To install the game onto your system, head to out -> make -> squirrel.windows -> x64 -> run spaceinvaders-1.0.0 Setup.exe.
